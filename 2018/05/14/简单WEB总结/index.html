简单WEB总结

2018-05-14 PV:

把一个阶段下来学到的web知识总结一下，简单地收个尾。

PHP黑魔法

接触了web后，最大的感触莫过于php真的是世界上最好（玩）的语言。稍微整理了一些简单的知识。

弱类型

弱类型的语言对变量的数据类型没有限制，你可以在任何地时候将变量赋值给任意的其他类型的变量，同时变量也可以转换成任意地其他类型的数据。

类型转换是无法避免的问题。例如需要将GET或者是POST的参数转换为int类型，或者是两个变量不匹配的时候，PHP会自动地进行变量转换。但是PHP是一个弱类型的语言，导致在进行类型转换的时候会存在很多意想不到的问题。

php函数

简单地记录几个函数。

strcmp()

任意数组通过strcmp与任意字符串比较的结果都是NULL。其他还有很多玩法。

md5()和sha()

这两个函数对数组加密会产生NULL。

extract()

变量覆盖。

ereg()

正则表达式函数ereg可通过%00截断绕过。

strpos()

strpos可通过数组返回NULL来绕过。

SQL注入

主要对普通回显注入、布尔盲注、时间盲注有一些经验。还有一些绕过的方式有了一些了解。

mysql基础函数

简短地记录几个。

if()和case when

用于时间盲注的判断。

sleep()和benchmark()

用于时间盲注。

order by

猜测select语句中的列数。

concat()和group_concat()

用分隔符连接字符串。

substr()和mid()

用于盲注时的判断。
