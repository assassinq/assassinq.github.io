Docker/Dockerfile基本命令

2018-04-22 PV:

Docker学习第一步。

Docker与虚拟机的优缺点

特性 Docker容器技术 虚拟机技术
占用磁盘空间 小，甚至只有几十KB（镜像层的情况） 非常大，可达GB
启动速度 快，几秒钟 慢，几分钟
运行形态 直接运行于宿主机的内核上，不同容器共同享同一个Linux内核 运行于Hypervisior上
并发性 一台宿主机可以启动成百上千个容器 最多几十个虚拟机
性能 接近宿主机本地进程 逊于宿主机
资源利用率 高 低
隔离性 安全隔离 完全隔离

Docker 命令

命令 作用
docker pull 获取image
docker build 创建image
docker images 列出image
docker run 运行container
docker ps 列出container
docker rm 删除container
docker rmi 删除image
docker cp 在host和container之间拷贝文件
docker commit 保存改动为新的image
docker diff 查看容器变化
docker events 查看事件
docker exec 进入容器
docker export 导出容器
docker history 查看镜像历史
docker import 导入容器
docker info 查看docker信息
docker inspect 查看各项详细信息
docker kill 杀死容器
docker login 登陆仓库
docker logout 登出仓库
docker logs 查看容器日志
docker network 管理网络
docker node 管理节点
docker pause 暂停容器
docker port 查看容器端口
docker rename 重命名容器
docker restart 重启容器
docker save 导出镜像
docker search 搜索镜像
docker service 管理服务
docker stats 查看容器状态
docker stop 停止容器
docker version 查看docker版本

Dockerfile 语法

命令 作用
FROM 导入基础镜像
RUN 执行命令
ADD 添加文件
COPY 拷贝文件
CMD 执行命令
EXPOSE 暴露端口
WORKDIR 指定路径
MAINTAINER 维护者
ENV 设定环境变量
ENTRYPOINT 容器入口
USER 指定用户
VOLUME 挂载点
ONBUILD 设置二次指令
LABEL 设置元数据
ARG 设置构建变量
STOPSIGNAL 设置停止信号
HEALTHCHECK 检查镜像状态
SHELL 设置命令执行环境
