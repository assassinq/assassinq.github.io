安装流程：

1.hexo 是基于 nodejs 的，需安装 nodejs，安装 nodejs 最好选择 homebrew

2.首先查看电脑是否安装 ruby，因为 homebrew 安装依赖 ruby

3.安装顺序：homebrew->nodejs->hexo
