2018-07-02 PV:

下午学姐介绍了一下burpsuite的基本功能，练习了几道题目。

Burpsuite相关功能

功能有点多有点写不动

这个网站有很多相关的介绍就不多解释了嘿嘿嘿

题目1

提示

burpsuite抓包后在http头中增加

或是在火狐插件中设置

然后再次得到提示

再在Accept-Language中删去除中文外的其他部分得到flag

题目2

由题目提示可知，密码为1000-9999的范围内。因为该处post有验证码，在burpsuite中进行爆破是最好的选择

这里最主要的是抓包时不能输错验证码

题目3

右键源码得到提示

登陆后在cookies中返回了一个提示newpage，显然是base64解密

打开新的网页后，一开始猜测是xss。然后随便输入留言内容，再抓包抓下来，发现没有权限。

尝试修改userlevel的值为root，没有权限的提示消失了

然后发现返回的set-cookies中islogin的值为0，尝试修改为1，即得到flag（%7B和%7D分别表示前后单引号）
