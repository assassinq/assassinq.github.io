变量规则

  • 变量以$符号开头
  • 变量名称必须以字母或下划线开头
  • 变量名称不能以数字开头
  • 变量名称只能包含字母数字字符和下划线（A-Z、a-z、0-9以及_）

MYSQL相关命令及注释

命令

  • AND：所有由AND连接的条件都为TRUE，SQL语句才执行
  • OR：只要OR连接的条件里有一个是TRUE，SQL语句就会执行
  • UNION SELECT：联合查询
  • GROUP BY：表示按照第几列进行排序
  • GROUP_CONCAT：将查询结果以一行的形式输出
