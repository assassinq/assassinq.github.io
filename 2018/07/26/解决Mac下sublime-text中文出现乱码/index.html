解决Mac下sublime text中文出现乱码

2018-07-26 PV:

用了这么久mac，竟然第一次发现sublime不能解码中文字体。

1、打开控制台

打开sublime，control+~打开控制台。

2、下载package control

如果是sublime text 2，输入：

如果是sublime text 3，输入：

详情见官网

3、下载ConvertToUTF8

重启sublime。command+shift+p，在输入框输入Package Control: Install Package，待载入所有的包目录后输入ConvertToUTF8，回车安装。

4、安装Codecs33

如果重启sublime之后仍然无法正确解码，需要下载Codecs33。将下载完的压缩包解压，把文件夹的名字修改为Codecs33并放在sublime的package目录下。

重启之后就大功告成啦。
