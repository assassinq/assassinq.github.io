下面做一些分解：

  • func1: {⍵(~⍵)/'success' 'fail'} // judge
  • func2: {+/⍵/⌽2*⍳⍴⍵} // bin_arr mul 2
  • func3: {a≠8↑(0,a←(8⍴2)⊤⍵)} 127 ==> 0 1 1 1 1 1 1 1 // f(x) = x ^ (x >> 1) // 2⊥0 1 1 1 1 1 1 1 ==> 127
  • part1: 'abc'{ {⍵(~⍵)/'success' 'fail'}⊃(+/⍺=⍵)=⍴⍺}'abc' ==> 'success'
  • part2: ⎕UCS // turn to ascii
  • part3: (-⌊(⍴'f0xtr0t')÷2) ==> -3 // -(len('f0xtr0t') / 2)
  • part4: 11 24⍴∊{part3} & 33 8⍴ // turn into a axb matrix
  • part5: ⌽⍉ // matrix transposition

加密的步骤：

  1. Convert to decimal
  2. Reverse the array
  3. Convert each value to an array of 8 bits and apply f(x) = x ^ (x >> 1)
  4. Reshape into 11x24
  5. Transpose and retate by -3
  6. Transpose and retate by 7
  7. Reshape into 33x8
  8. Convert to decimal and add 13
  9. Convert to ascii

reverse

解密函数：

分解部分：

  • func1: {2⊥a≠8↑(0,a←(8⍴2)⊤⍵)} // reverse f(x) = x ^ (x >> 1) <== fun1 ^ (N - 1) = func1 ^ (-1)

  • [x] func2: {(8⍴2)⊤⍵} // decimal to bits

  • [x] part1: ⌽ // reverse

  • part2: {}⍣7 // for 7 times

逆向的过程：

  1. Convert to decimal
  2. Substract 13
  3. Convert into an array of 8 bits
  4. Reshape into 11x24
  5. Reverse the transpose-and-reverse process
  6. Reshape into 33x8
  7. Convert to decimal
  8. Reverse the kind-of-xor process
  9. Convert to ascii
  10. Reverse the string
